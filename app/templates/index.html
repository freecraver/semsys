... consists of three layers and shows a world map with color coded countries according to their risk level on the first. The second layer shows capitals, the third is not selected by default and shows Ski Resorts. Clicking a country reveals country information such as the name, ISO3 code, etc. Clicking a capital marker shows weather information. In the select box below the map a month can be chosen. The map then displays towns inside the country that have a weather recommendation for the given month.

... is a visualization of which countries have articles where a search term is mentioned. The input field allows the user to perform an async typeahead search, which queries the triple store for matching resources mentioned in any news article. Keep in mind two limitations: The queries news results are from December 2019, which means any recent events are not covered. Also, no relation checks were performed, i.e. results only show the specific resource was mentioned in a news article, which also mentioned the specific country. This leads to problems for newsflash articles, which aggregate multiple different events.

... provides a selection of some of our SPARQL queries which were suited for visualization. The desired view can be selected via the dropdown box and the query is then directly executed on the server and visualized on our world map.
